# Resume-Job Matching

A simple tool that compares a resume with job descriptions and ranks the most relevant jobs using semantic similarity.

Group project by Hongrui Qu, Jiayi Li, Joohyun Park, Tracy Cui

## Tech

Python, Pandas, NumPy, scikit-learn, SentenceTransformers, Matplotlib, pdfplumber, Selenium
